Release v2.0.3 quick check
--------------------------

To try the released version:

git fetch --all --tags
git checkout v2.0.3

Task 0 is the scheduler role. It is enabled for nodes that accept template 0.
Only the node that owns the scheduler queue turn runs the validator-committee
prune. If no scheduled tasks are due, the node advances the scheduler queue
without opening and closing an empty scheduler round.
